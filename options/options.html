Kill Switch Toggle

If this key is pressed, all unproductive tabs will be closed. You can change the shortcut here

CTRL + Shift + K

Undo Kill Switch

If this key is pressed, all previously closed unproductive tabs will be re-opened. You can change the shortcut here

CTRL + Shift + U
